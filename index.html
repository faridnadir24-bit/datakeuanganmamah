📖 Panduan Setup Google Sheets

✅ Setup ini hanya dilakukan sekali saja. Setelah selesai, data akan tersimpan otomatis ke Google Sheets Mamah.
  1. Buka sheets.google.com → buat spreadsheet baru → beri nama "Kasir Mamah"
  2. Di dalam spreadsheet, klik menu Extensions → Apps Script
  3. Hapus semua kode yang ada, lalu paste seluruh isi file Code.gs yang sudah didownload
  4. Klik tombol 💾 Save, lalu klik Deploy → New deployment
  5. Pilih type: Web app
    Execute as: Me
    Who has access: Anyone
    Klik Deploy
  6. Google akan minta izin → klik Authorize access → pilih akun Google Mamah → klik Allow
  7. Copy URL yang muncul (bentuknya https://script.google.com/macros/s/XXXX.../exec)
  8. Kembali ke website ini → tempel URL tadi di kotak di atas → klik Hubungkan
  9. 🎉 Selesai! Data akan langsung tersimpan ke Google Sheets
⚠️ Penting: Jika muncul "Google hasn't verified this app", klik Advanced → Go to Kasir Mamah (unsafe). Ini aman karena scriptnya milik Mamah sendiri.
